# Nuxt.js

## Introduction
The following guide describes how to integrate Tiptap with your [Nuxt.js](https://nuxtjs.org/) project.

## Requirements
* [Node](https://nodejs.org/en/download/) installed on your machine
* Experience with [Vue](https://vuejs.org/v2/guide/#Getting-Started)

## 3. Create a new component
To actually start using Tiptap, youâ€™ll need to add a new component to your app. Letâ€™s call it `TiptapEditor` and put the following example code in `components/TiptapEditor.vue`.

```html

  <editor-content :editor="editor" />
</template>

import StarterKit from '@tiptap/starter-kit'

  components: {
    EditorContent,
  },

  data() {
    return {
      editor: null,
    }
  },

  mounted() {
    this.editor = new Editor({
      content: '<p>Iâ€™m running Tiptap with Vue.js. ðŸŽ‰</p>',
      extensions: [
        StarterKit,
      ],

  },

    this.editor.destroy()
  },
}
</script>
```

## 4. Add it to your app
Now, letâ€™s replace the content of `pages/index.vue` with the following example code to use our new `TiptapEditor` component in our app.

```html

  <div>
    <client-only>
      <tiptap-editor />
    </client-only>
  </div>
</template>
<script>
import TiptapEditor from '~/components/TiptapEditor.vue'
export default {
  components: {
    TiptapEditor
  }
}
</script>
```

Note that Tiptap needs to run in the client, not on the server. Itâ€™s required to wrap the editor in a `<client-only>` tag. [Read more about client-only components.](https://nuxtjs.org/api/components-client-only)

You should now see Tiptap in your browser. Time to give yourself a pat on the back! :)

## 5. Use v-model (optional)
Youâ€™re probably used to bind your data with `v-model` in forms, thatâ€™s also possible with Tiptap. Here is a working example component, that you can integrate in your project:

https://embed.tiptap.dev/preview/GuideGettingStarted/VModel

```html
<template>
  <editor-content :editor="editor" />
</template>

<script>
import { Editor, EditorContent } from '@tiptap/vue-2'
import StarterKit from '@tiptap/starter-kit'

export default {
  components: {
    EditorContent,
  },

  props: {
    value: {
      type: String,
      default: '',
    },
  },

  data() {
    return {
      editor: null,
    }
  },

  watch: {
    value(value) {
      // HTML
      const isSame = this.editor.getHTML() === value

      // JSON
      // const isSame = JSON.stringify(this.editor.getJSON()) === JSON.stringify(value)

      if (isSame) {
        return
      }

      this.editor.commands.setContent(value, false)
    },
  },

  mounted() {
    this.editor = new Editor({
      content: this.value,
      extensions: [
        StarterKit,
      ],
      onUpdate: () => {
        // HTML
        this.$emit('input', this.editor.getHTML())

        // JSON
        // this.$emit('input', this.editor.getJSON())
      },
    })
  },

  beforeDestroy() {
    this.editor.destroy()
  },
}
</script>
```
