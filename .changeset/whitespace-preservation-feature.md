Add `preserveWhitespace` option to Heading and Paragraph extensions for preserving tabs and multiple spaces in rendered HTML output. When enabled, applies 'white-space: pre-wrap' CSS styling and sets the node's whitespace property to 'pre', allowing tabs, spaces, and newlines to be preserved as-is in the editor.
