Add new Decorations API that allows creating decorations directly from extensions. This API abstracts ProseMirror's decoration complexity and provides three types of decorations: inline decorations for styling text ranges, node decorations for styling entire nodes, and widget decorations for inserting custom components. Includes `WidgetRenderer` utilities for React, Vue 2, and Vue 3 frameworks. [Read more](https://tiptap.dev/docs/editor/core-concepts/decorations)
